# ShiftClock

一个简单易用的排班查看器，帮助您轻松管理工作时间。支持自定义工作时长和休息时长，自动计算排班周期，实时显示当前工作状态。提供清晰的日历视图和详细的排班列表，让您随时掌握工作安排。界面采用现代化设计，支持单休、双休和自定义休息日设置，满足不同工作模式的需求。

"本项目由阿里云ESA提供加速、计算和保护"
![阿里云ESA](./images/image.png)

## 功能特点

- 📅 **灵活的排班设置**：支持自定义工作时长和休息时长
- 🕐 **实时状态显示**：显示当前是上班还是休息状态
- 📋 **详细排班表**：清晰展示未来一段时间的排班安排
- 🎨 **现代化界面**：美观的渐变设计和响应式布局

## 使用方法

1. 设置工作时长（例如：12小时）
2. 设置休息时长（例如：24小时）
3. 选择起始时间
4. 设置要查看的天数
5. 点击"计算排班"按钮

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```
